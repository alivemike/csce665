CSCE 665 Homework 1

You are required to finish this assignment individually.

Write a program to reassemble telnet, ftp, and HTTP sessions. Your program should display for each telnet, ftp, and HTTP session, in ASCII format, the command/response interactions between the client and the server. You may use C, C++, or Java for this assignment.
 * The input is a network trace. A sample pcap trace with telnet and ftp sessions is available here for your test. A sample HTTP trace is here. We may use another trace (ftp/telnet/http) to test your submitted program.
 * The interactions you need to display is NOT at packet level, but application level.
 * The output format is up to you; however, the output must be ascii readable, so convert any unreadable characters to some numeric value (e.g., %u, %d). And you may need to combine/assemble multiple packets in a single response.
 * You may need to consult the relevant RFCs for the tcp/telnet/ftp/http protocols.
 * You can use libpcap library in your program (more tutorial here, here, and here; java version here or here). You can refer to existing open source projects (e.g., tcpdump, snort, bro) and modify them according to your purpose.
 
Your submission should include:
    * Compilable source code of your solution for the assignment.
    * A report describing everything I should know to be able to test your solution, as well as detailed design, debug, experience, any of your finding/conclusion, and any thoughts, lessons.
